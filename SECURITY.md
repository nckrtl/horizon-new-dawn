# Security Policy

Horizon New Dawn is currently intended for development and testing environments. The project does not yet provide production-readiness guarantees or a formal security-support window.

Please do not open public issues for suspected vulnerabilities. Report them privately through [GitHub's security advisory form](https://github.com/nckrtl/horizon-new-dawn/security/advisories/new) and include the affected version, impact, reproduction steps, and any suggested mitigation.

Security reports will be acknowledged as soon as practical. Until a fix is available, avoid publishing details that could put users at risk.
